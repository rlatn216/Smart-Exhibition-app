알파벳 블록을 활용한 영어 학습 앱

  • ○ 프로젝트 소개
    • 영어에 대한 아이의 자율학습 및 재미를 충족하는 애플리케이션 개발
    • 알파벳 블록들을 직접 배열하여 앱의 영상처리 기능을 통해 단어의 뜻을 알려주는 시스템
    • 블록을 사용해서 일종의 놀이를 통하여 영어에 대한 접근성을 높여주는 프로젝트

  • ○ 개발배경 및 필요성
    • 밀레니얼, 즉 젊은 세대의 부모님들은 아이들의 조기교육에 관심이 높음
    • 애플리케이션과 알파벳 블록을 통해 학습하기에 기존의 학습 장난감보다 비용 절감
    • 아이들이 재미를 느끼는 학습 놀이를 통해 영어에 재미를 붙임으로써 자발적인 공부에 도움이 됨

  • ○ 개발목표
    • 영어단어를 한국어로 한국어를 영어로 번역해주는 서비스 API 이용
    • 애플리케이션에 인식된 번역된 문자를 음성으로 출력해주는 애플리케이션 개발
    • 영상처리를 통해 문자를 인식해주고 다른 배경이 찍혀도 무관하게 문자를 인식해주는 기능 개발

  • ○ 구성도
      프로젝트 흐름도

    • Machine Learning을 이용한 텍스트 인식을 통해 블록에 적힌 문자를 인식
    • 인식된 문자 데이터를 애플리케이션에 받아 번역해주는 API 서비스에 전달
    • API에서 전달받은 문자 데이터를 한국어는 영어로 영어는 한국어로 번역
    • 번역한 문자들을 Speech API를 통해 스마트폰으로 음성 출력

  • ○ 작품 소개 영상 및 시연 영상
